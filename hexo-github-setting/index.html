建立自己Blog系列(二) github Setting

Posted on 2019-06-02 | Edited on 2021-10-16 | In WebDesign | Comments: | Views:
Symbols count in article: 801 | Reading time ≈ 1 mins.

在Github這邊要先建立兩個Repository

  1. Blog Code
  2. Blog Hosting

Blog Code

這個Repository是利用Hexo cli樣板來撰寫Blog.

像我目前使用的MyBlog

建立完自己的Code Repository後.

將他clone到自己電腦上面.

執行下面幾個步驟

  1. 打開此Repository資料夾
  2. 打開CMD並進入此資料夾
  3. 利用NodeJS NPM 打 npm install hexo -g 就會開始下載
  4. 執行完第三步後再打Hexo init指令,安裝看建立Hexo將檔案.

Blog Hosting

在建立另一個Repository來當html blog靜態託管使用.

就像石頭使用isdaniel.github.io

執行下面幾個步驟

  1. 建立一個Repository 名子要取為 [username].github.io，其中的[username]是您Github帳號.
  2. 進入此Repository中，並進入Setting
  3. 確認是否已經成功將此Repository Hosting在你的Github page上
    如果成功呈現此圖

我們可以嘗試在這個Hosting Repository建立一個index.html 在上面打Hello would並commit push

在訪問https://[username].github.io/應該就可以看到Hello Would文字.
